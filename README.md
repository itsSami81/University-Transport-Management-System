**University Transport Management System**

A console-based University Transport Management System developed in C++ as an Object-Oriented Programming (OOP) semester project.

The system provides transport management functionalities for both students and administrators, including student registration, transport applications, route and vehicle management, transport pass generation, billing, fee payment, application approval/rejection, and report generation.

**Features**

Student & Admin Authentication
Route Management
Bus & Van Management
Transport Pass System
Billing & Late Fine Management
Vehicle-to-Route Assignment
File Handling for Data Persistence
Report Generation
